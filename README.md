 # fzp-cli
![Npm 版本](https://img.shields.io/badge/auto-cli_v0.01-green)
 ## 简介
 -用于快速搭建前端项目的命令行工具

 ## 模板
 -快速生成  Vue3 + TS+vite+elementPlus+Pinia+Axios+Mock的模板

 ## 安装

 ```bash
npm install fzp-cli -g
```

## 使用

```bash
#使用模板创建项目
fzp create

#执行项目文件夹名称并选择模板创建新项目
fzp create [projectName]

#查看脚手架版本
fzp -v
